🚀 HireSphere – Full Stack MERN Job Portal

A production-ready Job Portal built using the MERN stack with role-based authentication, resume upload system, recruiter dashboard, and applicant tracking features.

🌐 Live Demo:
👉 https://job-portal-mern-mu.vercel.app

📌 Overview

HireSphere is a complete job management platform where:

👩‍💻 Applicants can browse jobs, apply with resume (PDF), and track application status.

🧑‍💼 Recruiters can post jobs, view applicants, review resumes, and update application status.

The project demonstrates real-world full-stack architecture, authentication, file handling, deployment, and database integration.
